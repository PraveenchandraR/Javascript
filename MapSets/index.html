Question 1.  You are given a string (STR) of length N, consisting of only the lower case English alphabet.
        Your task is to remove all the duplicate occurrences of characters in the string.


        Input- abcadeecfb
        output- output has to be abcdef

        A.  function RemoveDuplicates(str) {
              let alphabets = new Set();
              let output = '';
        
              for (let i = 0; i < str.length; i++) {
                  if (!alphabets.has(str[i])) {
                      alphabets.add(str[i]);
                      output= output+ str[i];
                  }
               }
        
               return output;
          }
